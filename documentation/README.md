# Documentation

Documentation and supporting materials for the EmergencyIQ dissertation and system design.

Subfolders:
- dissertation/: Thesis drafts, chapters, and figures.
- research_papers/: Papers and related notes.
- architecture/: Architecture notes and decisions.
- diagrams/: High-level diagrams and exported images.
- UML/, ERD/, DFD/: Design artifacts and diagrams.
- API/: API specifications and example usage.
- meeting_notes/: Project meeting minutes and action items.
- references/: Bibliography and citations.

Keep documentation versioned and cite sources where appropriate.
