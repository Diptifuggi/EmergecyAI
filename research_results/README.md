# Research Results

Results and artifacts from experiments, including metrics, plots, logs, and benchmarks used in the dissertation.

Subfolders:
- accuracy/, confusion_matrix/, roc_curves/, precision_recall/: Evaluation metrics and plots.
- training_logs/: Raw training logs and experiment traces.
- benchmark/: Benchmark results and comparison tables.
- latency/, memory_usage/: Resource profiling results.
- inference_results/: Example inferences and outputs.
- graphs/, tables/: Visualizations and tabular summaries.

Keep this directory organized by experiment id / date to preserve reproducibility.
