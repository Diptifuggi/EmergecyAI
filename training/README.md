# Training

This directory contains notebooks, scripts, and configuration used to train and evaluate models for EmergencyIQ.

Subfolders:
- notebooks/: Jupyter notebooks for experiments and analysis.
- preprocessing/: Data cleaning and preprocessing utilities.
- feature_engineering/: Feature extraction and selection code.
- augmentation/: Data augmentation pipelines.
- configs/: Training and hyperparameter configuration files.
- scripts/: Training and evaluation scripts.
- utilities/: Helper utilities and common functions.
- checkpoints/: Model checkpoints and intermediate artifacts.
- logs/: Training logs and experiment traces.

Keep code modular to allow reproducible experiments.
