# Datasets

This directory contains public and private datasets used for research and training in EmergencyIQ.

Structure:
- raw/: Original, unmodified datasets (audio, transcripts, labels).
- processed/: Cleaned and preprocessed data ready for model training.
- train/, validation/, test/: Splits used for experiments.
- metadata/: Dataset descriptions, labels, and provenance information.

Do NOT store sensitive raw data without anonymization or proper access controls.
